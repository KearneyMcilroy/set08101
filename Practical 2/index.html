Choose Your own Adventure Game - SET08101

The creation of this website has been done as it was a task for the Web Technologies Module (SET08101). I have created a Choose your Own Adventure Game using HTML. I thoroughly enjoyed creating this game as it made me have to think out of the box as I have had to think of some form of storyline. My game is based in a fantasty medieval time period.
